# HelpUp

`Análise e Desenvolvimento de Sistemas - PUC Minas`

`Desenvolvimento de uma Aplicação Móvel`

`2° semestre/2025`

O projeto Help Up consiste no desenvolvimento de um aplicativo móvel voltado à conexão entre voluntários e organizações sociais, com o objetivo de centralizar e facilitar o acesso a oportunidades de trabalho voluntário no Brasil. A plataforma busca otimizar a comunicação e a gestão das atividades, permitindo que voluntários encontrem ações alinhadas aos seus interesses, disponibilidade e localização, enquanto as instituições podem divulgar suas necessidades de forma prática e eficiente.

Com uma interface intuitiva e recursos que permitem o cadastro, acompanhamento e avaliação das experiências, o Help Up pretende fortalecer a cultura do voluntariado, ampliar o engajamento social e maximizar o impacto das ações solidárias. A iniciativa utiliza a tecnologia como ferramenta de inovação social, aproximando pessoas e organizações e contribuindo para o fortalecimento das comunidades e o bem-estar coletivo.
